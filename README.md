# Installation
* 1 - clone repo https://github.com/krstaslan/Devlinking
* 2 - create a virtual environment and activate
*  - pip install virtualenv
*  - virtualenv envname
*  - envname\scripts\activate
* 3 - pip install -r requirements.txt
* 4 - python manage.py runserver
